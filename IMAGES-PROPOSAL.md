# Image Sourcing Proposal

One candidate per placeholder, with a one-line backup where it helps. All candidates are public-domain or clearly CC-licensed for educational reuse, except where flagged (and those are flagged in **bold**).

License key:
- **PD** — public domain
- **PD-Bain** — Library of Congress / George Grantham Bain Collection, no known publication restrictions
- **PD-USGov** — US federal government work
- **CC-BY 2.0** — Creative Commons Attribution; requires attribution line in caption
- **CC-BY-SA** — Attribution + share-alike; same requirement
- **CERN** — CERN copyright but typically granted free educational use; ⚠ verify
